# Project2-Wireshark-Traffic-Analysis
## Objective:
Capture and analyze live network traffic to identify active protocols.connections,timestamps and any potential suspicious activity.

## Tools Used:
Wireshark
Virustotal

## What i did:
I was able to capture live Wifi traffic on my network.
I also filtered traffic using protocols(DNS-Domain Name System and TCP- Transmission Control Protocol)
Identified active applications communicating over the network

## Protocols Obsereved:
TCP- which ensures relaible data transfer between devices
DNS- It translates domain names to IP addresses
SSDP- it discovers devices on a local network
TLSv1.2 - it ensures theres encrypted secure information

## DNS Analysis
Domains observed in the traffic include:
spotify.com - spotify streaming music
microsoft.com - microsoft cloud
whatsapp.net - running whatsapp 

## Key findings
Learned that DNS monitoring is critical for early detection on threats
All personal network traffic was clean with no suspicious activity going on

## What i learned:
i learned how to capture and filter live traffic in wireshark
I learned to read specific packet details like the source ip and destination ip,ports used and also the protocols invloved.
Learned what the beaconing behaviour is and how it acts as malware indicator.
I also learned to investigate IPs using Virustotal whether genuine or suspicious.
